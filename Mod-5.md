
# Exp No - 5

## TITLE : Access Modifiers

### AIM : To demonstrate the use of a final variable in Java.

### ALGORITHM :

STEP 1 : Start

STEP 2 : Define a class with a final variable universityName

STEP 3 : Create an object of the class

STEP 4 : Access and print the final variable

STEP 5 : End

### PROGRAM :
```
class College {
    final String universityName = "Saveetha University";
}

class prog {
    public static void main(String[] args) {
        // Create object of College
        College c = new College();

        // Print the university name
        System.out.println(c.universityName);
    }
}
```
